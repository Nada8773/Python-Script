# Requirment

#### Linear Merge
```
Given two lists sorted in increasing order, create and return a merged
list of all the elements in sorted order. You may modify the passed in lists.
- e.g. ['aa', 'xx'] and ['bb', 'cc', 'zz'] => ['aa', 'bb', 'cc', 'xx', 'zz']
```

#### Long Word
``` 
 Given a string and length of word n, return a list that contain only words 
 of length n or greater of the given string.
- e.g. long_words(3,"ITI is the best") -> ['ITI' , 'the' , 'best']
```
