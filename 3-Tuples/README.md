# Tuples

#### Given
```
 - a list of non-empty tuples
```
#### Requirment
```
- create script to sort_sum

    - e.g. [(1, 7), (1, 4), (3, 4, 5), (2, 2)] yields
    -      [(3, 4, 5), (1, 7), (1, 4), (2, 2)]
```
